Watchfolio API documentation

This api updates every day. By using puppeteer.js, sold listings from eBay and Chrono24 are scraped and a mean is calculated. This mean is then stored in a csv file that can be found here. The price property of each watch is derived from this mean and csv file.

Start here

To start using this api, direct fetch requests to watchfolio.com/api/watches to get json response containing:

  • watches: property which contains an array of watch objects.each watch is ordered defaultly from lowest price to highest price, irrespecive of brand. each watch object looks like:
    {
    name: Omega Seamaster Tokyo Edition (string)
    brand: Omega (string)
    model: 2020 (string)
    price: 4384.83 (double)
    7d_price_move: -2.3% (string)
    30d_price_move: -4.8% (string)
    1yr_price_move: 21% (string)
    }
  • lastUpdated: property which contains the date in which data was last updated

Filtering

To filter watch by brand, direct fetch request to watchfolio.com/api/watches?brand=brandNameAsString keep in mind the supported brands are Rolex, Patek Philippe, Omega, Seiko, Cartier, IWC, TAG Heuer, querying any other brand will result in a 404 response.
